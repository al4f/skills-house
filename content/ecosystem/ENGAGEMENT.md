## Suggested discussion topics

Post these constructively where agentskills.io community channels exist:

### Topic 1: Source vs dist separation

> The Agent Skills spec defines dist well. I'd love to see optional guidance for authoring layouts with a build step — skills-house documents one approach: github.com/al4f/skills-house/specs

### Topic 2: Cross-skill dependency notes

> When skill A depends on skill B, a standard agent-facing install note would help. skills-house injects a Depends-on block at build time — happy to share the format if useful for spec v2.
